# Around – Interactive JavaScript Application

## Description

Around is an interactive web application that allows users to edit their profile and interact with dynamic content. The project focuses on JavaScript fundamentals such as DOM manipulation and event handling.

## Features

- Editable user profile with popup forms
- Dynamic content updates using JavaScript
- Event listeners for user interactions
- Form handling and UI updates

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
